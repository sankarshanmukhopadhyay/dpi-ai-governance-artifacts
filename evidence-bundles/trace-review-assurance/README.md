
# TRACE review assurance bundle

This example bundle shows the minimum connective tissue needed to move from a review output to an assurance-ready package.

Included artifacts:

- `review-bundle.json` — portable review export produced by the Lab
- `artifact-control-map.json` — explicit mapping from artifacts to control IDs
- `minimum-audit-trail.json` — minimal change and handling log
- `trace-conformance-declaration.json` — bounded declaration of what is being claimed

The point is not maximal bureaucracy. The point is to make it possible to test what a bundle claims to cover.
