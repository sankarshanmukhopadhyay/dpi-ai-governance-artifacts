# Reference Assurance Evidence Bundles

This directory provides a minimal layout for an assurance evidence bundle that can travel between implementation teams, reviewers, and procurers.

A valid minimal bundle includes:

- `governance_policy.md`
- `risk_register.json`
- `model_documentation.json`
- `evaluation_results.json`
- `incident_response_plan.md`

The point is not ceremonial completeness. The point is to make governance claims inspectable.
