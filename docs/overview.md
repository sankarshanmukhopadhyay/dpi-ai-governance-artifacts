# Documentation Overview

This repository contains **operational packs**: portable schemas, templates, playbooks, and conformance materials for DPI + AI governance.

## Start here

- `README.md` for the "what and why"
- `packs/README.md` for the pack catalog
- `REFERENCES.md` for upstream standards and sources
- `controls/` for machine-readable control registries (JSON/YAML)

## Navigation map

| Area | What it contains | Who uses it |
|---|---|---|
| `packs/` | Pack-level READMEs + artifacts | Implementers, program teams |
| `controls/` | Control objectives and registry | Assurance, audit, governance |
| `annex/` | Annex templates and structured materials | Policy and governance teams |
| `docs/` | Repository documentation spine | Everyone |
