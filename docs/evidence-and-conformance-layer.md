
# Evidence and conformance layer

This increment completes the minimum evidence-and-conformance connective tissue called for in the roadmap.

New components:

- `schemas/artifact-control-map.schema.json`
- `schemas/trace-conformance-declaration.schema.json`
- `schemas/minimum-audit-trail.schema.json`
- `evidence-bundles/trace-review-assurance/`

Together these make it possible to answer four basic questions:

1. What exactly is being claimed?
2. Which controls does the bundle map to?
3. What evidence is attached to that claim?
4. What audit trail exists for the handoff?
