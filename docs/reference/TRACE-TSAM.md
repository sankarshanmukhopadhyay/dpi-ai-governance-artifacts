# TRACE ↔ TSAM (Reference)

This repository contains **operational artifacts** for DPI AI governance.

The normative relationship between **TRACE** (governance lens) and **TSAM** (assurance engineering method) is defined in the **DPI AI Governance Lab** repository.

This repository **MUST NOT** redefine TRACE or TSAM. It provides **templates and machine-readable scaffolding** that instantiate the TRACE↔TSAM binding in deployable packs.

- DPI AI Governance Lab: `docs/trace/TRACE-TSAM.md` (forward reference)
- Canonical method spine: TRQP Assurance Hub: `docs/strategy/TRACE-TSAM-relationship.md`
